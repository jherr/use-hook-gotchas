# use hook gotchas

Run examples using a package manager

```sh
pnpm lazy-use
pnpm staggered
pnpm state-resets
```

or `npx @lazarv/react-server`

```sh
npx @lazarv/react-server@latest ./lazy-use.jsx
npx @lazarv/react-server@latest ./staggered.jsx
npx @lazarv/react-server@latest ./state-resets.jsx
```
